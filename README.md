# Football Science

📌 **Descrição**  
O **Football Science** é um projeto de **Ciência de Dados aplicada ao futebol**, focado em previsões de desempenho, modelagem estatística e aprendizado de máquina para analisar padrões e tendências.

🧠 **Principais Objetivos**  
- Criar modelos preditivos para prever classificações.  
- Relacionar investimentos e desempenho esportivo.
- Utilizar **machine learning** para identificar padrões ocultos.  

📌 **Tecnologias Utilizadas**  
- **Machine Learning:** scikit-learn, TensorFlow, XGBoost, LightGBM, CatBoost  
- **Análise de Dados:** Pandas, NumPy, Matplotlib, Seaborn  
- **Bases de Dados:** Transfermarkt, Sofascore, CBF  
- **Dashboards:** Power BI, Streamlit  

📂 **Estrutura do Projeto**  
```
Football_Science/
│── data/           # Conjunto de dados para treinamento de modelos
│── models/         # Modelos treinados e otimizados
│── notebooks/      # Notebooks de experimentação
│── src/            # Scripts para pré-processamento e modelagem
│── README.md       # Documentação do projeto
```

🚀 **Como Executar**  
1. Clone este repositório.  
2. Instale as dependências: `pip install -r requirements.txt`  
3. Execute os notebooks na pasta `/notebooks`.
